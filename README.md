# Monvex

Monvex is a fast currency converter and calculator designed for quick checks, clean interactions and a lightweight mobile experience.

![Platform](https://img.shields.io/badge/platform-Android-blue)

## Overview

Monvex focuses on speed and clarity. The app combines live exchange-rate conversion with a simple calculator, making it practical for travel, shopping and everyday financial checks.

## Key Features

- Live currency conversion
- Built-in calculator for quick comparisons
- Lightweight interface with a strong mobile-first layout
- Local history for recent conversions
- Multilingual foundation

## Privacy

The public privacy policy is available in [`privacy.html`](privacy.html).

This project is intentionally designed with privacy in mind:

- No account creation flow
- No analytics SDKs
- No personal-profile storage
- Network requests only for exchange-rate lookup and language detection

## Public Repository Setup

1. Keep `privacy.html` at the repository root.
2. Enable GitHub Pages from the `main` branch.
3. Use the Pages URL as the privacy policy URL in Google Play Console.
