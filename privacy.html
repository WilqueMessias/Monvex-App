Monvex Privacy Policy

Política de Privacidade

Esta política explica, de forma direta, quais dados o Monvex utiliza para funcionar, o que permanece no dispositivo e o que é enviado a serviços externos para consulta de câmbio.

Última atualização: julho de 2026 Versão do app: internal-test-1 Plataforma: Android

1. Introdução

O Monvex é um conversor de moedas e calculadora preparado para Android e Google Play. Esta página descreve como os dados são tratados durante o uso do aplicativo.

2. Informações coletadas

2.1 Dados não coletados

O Monvex não coleta os seguintes dados pessoais ou sensíveis:

  • Nome ou dados pessoais
  • E-mail ou telefone
  • Localização exata ou aproximada
  • Fotos, vídeos ou arquivos
  • Contatos ou calendário
  • Histórico de navegação
  • Dados biométricos

2.2 Dados implícitos em requisições de rede

Ao buscar taxas de câmbio em tempo real, o app faz requisições HTTPS para APIs de terceiros. Essas requisições podem incluir endereço IP do dispositivo e User-Agent.

Esses dados são transmitidos diretamente para os servidores da API e não são armazenados pelo Monvex.

2.3 Dados locais

O app armazena localmente no seu dispositivo:

  • Histórico de conversões realizadas
  • Preferências de idioma
  • Estado de assinatura, se aplicável

Esses dados permanecem exclusivamente no seu dispositivo e não são enviados para servidores do Monvex.

3. Como usamos as informações

  • Fornecer funcionalidade de conversor de moedas
  • Exibir taxas de câmbio atualizadas
  • Manter histórico local de conversões
  • Melhorar a compatibilidade do app com seu dispositivo

4. Compartilhamento de dados

Resumo de compartilhamento
  • Não vendemos ou compartilhamos dados pessoais.
  • IP e User-Agent podem ser compartilhados apenas com provedores de API de câmbio para funcionamento do app.
  • Não integramos analytics ou bibliotecas de terceiros para rastreamento de dados pessoais.

5. Segurança

  • Todas as requisições de API usam criptografia HTTPS/TLS.
  • Dados locais são armazenados de forma segura no seu dispositivo.
  • Não mantemos cópias de dados do usuário em servidores do Monvex.

6. Exclusão de dados

Histórico local: limpe através de:

  • Botão de limpar dentro do app
  • Configurações Android → Aplicativos → Monvex → Armazenamento → Limpar cache
  • Desinstalação do app

O provedor da API pode manter logs conforme a própria política. O Monvex não mantém backups dos dados do usuário.

7. Direitos do usuário

  • Acessar os dados locais a qualquer momento
  • Limpar o histórico dentro do app
  • Desinstalar o app a qualquer momento
  • Não ser rastreado por terceiros pelo Monvex

8. Conformidade legal

Esta política foi redigida para ser compatível com:

  • GDPR
  • COPPA
  • LGPD
